# URL Shortener

A Spring Boot project for building a small URL shortener.

The application will let a user save a long URL and get back a short URL that redirects to the original address.

## Features

- create a short URL from a long URL
- redirect a short code to the original URL
- keep the code clean and easy to understand

## Tech Stack

- Java 25
- Spring Boot 4
- Maven

## Run The App

```bash
./mvnw spring-boot:run
```

The app starts on:

```text
http://localhost:8080
```

## Run Tests

```bash
./mvnw test
```
